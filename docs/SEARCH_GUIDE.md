# Search Bar Guide

## What Can You Search?

The search bar on the **Discover** page allows you to search for recipes by:

### ✅ Searchable Fields

1. **Recipe Title** - Search by recipe name
   - Example: "pasta", "chicken", "cake"

2. **Description** - Search in recipe descriptions
   - Example: "Italian", "spicy", "healthy"

3. **Ingredients** - Search by ingredient names
   - Example: "tomato", "cheese", "flour"

4. **Tags** - Search by recipe tags
   - Example: "vegetarian", "dessert", "quick"

### 🔍 How to Use

1. **Type your search term** in the search bar
2. **Results update automatically** as you type
3. **Search works across:**
   - User-created recipes in the database
   - TheMealDB recipes (external API)

### 📝 Search Examples

- **"pasta"** - Finds all pasta recipes
- **"chicken"** - Finds recipes with chicken in title, description, or ingredients
- **"chocolate"** - Finds chocolate desserts and recipes
- **"italian"** - Finds Italian cuisine recipes
- **"quick"** - Finds quick/easy recipes

### 💡 Tips

- Search is **case-insensitive** - "Pasta" and "pasta" return the same results
- Search matches **partial words** - "past" will find "pasta"
- You can combine search with **filters** (category, cuisine)
- Clear search by clicking "Clear all filters" button

### 🔧 Technical Details

- Searches both **local database recipes** and **TheMealDB API**
- Results are combined and displayed together
- Search is performed in real-time as you type
- Maximum 20 results per page with pagination

---

**Note:** If search doesn't return results, try:
- Using different keywords
- Checking your spelling
- Using broader terms (e.g., "chicken" instead of "chicken breast")
- Clearing filters and trying again
